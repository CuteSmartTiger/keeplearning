首先使用pycharm创建Django项目
然后配置pycharm中的虚拟环境
然后使用pycharm安装需要的包



Django项目的启动：
1. 先启动activate.bat脚本
2. python manage.py runserver



配置数据库：
```python
DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.mysql',  # 数据库引擎
        'NAME': 'desktop',  # 你要存储数据的库名，事先要创建之
        'USER': 'root',  # 数据库用户名
        'PASSWORD': '123123',  # 密码
        'HOST': '192.168.6.95',  # 主机
        'PORT': '3306',  # 数据库使用的端口
        # 'OPTIONS': {'init_command': 'SET storage_engine=INNODB;'}  # 初始化数据库操作，指定数据库引擎
    }
```
